Welcome!¶

Welcome to the documentation of the Small Vehicles for Autonomy (SVEA) platform! Currently, the documentation is focused towards giving an overview of the different modules and functionalities provided by the svea_starter workspace. For instructions on how to install and use the different tools in the svea_starter workspace, please refer to the README on the Github repo.
