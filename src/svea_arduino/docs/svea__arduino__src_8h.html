void processPwm ( )

Handle pwm events.

Reads all the values from PWM_T_BUFFER and PWM_S_BUFFER that has been added since the last loop and translates the pwm signal to actuation values. Calls pwmEvent() 3 ms after the last observed rising edge.

Button value directions: Ctrl / Value Throttle pulled -> Possitve value Steering forward -> Possitive value Gear switch up -> 0 (Low gear) Gear switch down -> 1 (High Gear) Switch back -> forward diff 0 (unlocked), back diff 0 (unlocked) Switch middle -> forward diff 1 (locked), back diff 0 (unlocked) Switch middle -> forward diff 1 (locked), back diff 1 (locked) and REM_OVERRIDE = true
